# Projeto Nota 10 - GenAI, Dados & Cyber

## Contexto e Objetivos
Este projeto faz parte do Bootcamp DIO e tem como foco os temas **GenAI, Dados e Cyber**.  
O objetivo é consolidar conhecimentos técnicos, praticar engenharia de prompts e criar um miniguia de estudo que possa servir como referência futura para revisões e aplicações práticas.

## Engenharia de Prompts e Cicatrizes
Durante o estudo, foram elaborados e testados diferentes prompts para explorar os temas.  
Documentação inclui:
- **Prompt inicial:** exemplo de pergunta feita à IA  
- **Resposta obtida:** resumo da resposta gerada  
- **Ajustes realizados:** como o prompt foi modificado para melhorar a saída  
- **Dificuldades encontradas:** pontos de atenção e troubleshooting  

## Miniguia de Estudo
### Resumos Estruturados
- Principais conceitos de GenAI
- Fundamentos de dados e análise
- Noções de segurança cibernética

### Glossário
- **GenAI:** definição e aplicações  
- **Big Data:** conceito e importância  
- **Cybersecurity:** práticas e ferramentas  

### Prompts Reutilizáveis
- “Explique o conceito de GenAI em linguagem simples.”  
- “Liste aplicações práticas de análise de dados em empresas.”  
- “Quais são as principais ameaças em segurança cibernética atualmente?”

## Como Entregar
1. Estruture este README.md com seu conteúdo.  
2. Adicione as fontes na pasta `fontes`.  
3. Documente os testes de prompts na pasta `prompts`.  
4. Coloque o miniguia consolidado na pasta `miniguia`.  
5. Copie a URL do repositório e entregue na plataforma da DIO.
